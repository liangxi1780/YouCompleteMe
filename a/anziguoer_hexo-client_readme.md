### 功能简介
- 文章添加
- 文章修改
- 文章删除
- 文章支持七牛图片上传
- 支持hexo generate
- 支持hexo deploy

### Build Setup

``` bash
# install dependencies
npm install

# dev
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
