## Quick Start

```html
 
 

 
     Quick Start 
      
 

 
     
        const MyApp = san.defineComponent({
            template: `
                 
                     
                     Hello {{name}}! 
                 
            `
        });

        let myApp = new MyApp({
            data: {
                name: 'San'
            }
        });
        myApp.attach(document.body);
     
 

 
```
