# develop
for devloper, need node.js and electron installed, then:

* npm install
* npm start

## build
before build:
set 

```
ELECTRON_MIRROR=http://npm.taobao.org/mirrors/electron/
```

for windows:
```
npm run package-x64
```
or
```
npm run package-x86
```

for osx package(must under osx):
```
npm run package-osx
```
