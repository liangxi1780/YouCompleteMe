## 安装

使用composer安装

~~~
git clone https://gitee.com/bool_it/bar_management_system.git
~~~

启动服务

~~~
cd tp
php think run
~~~

然后就可以在浏览器中访问

~~~
http://localhost:8000
~~~

更新框架
~~~
composer update topthink/framework
~~~
