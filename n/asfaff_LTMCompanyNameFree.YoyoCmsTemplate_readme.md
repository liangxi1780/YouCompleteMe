# 前端项目的说明

**如何阅读文档**

在开始之前有一些文档描述约定说明，这有助于更好的阅读：

- API 相关
  - `[]` 表示属性
  - `()` 表示事件
  - `[()]` 表示双向绑定
  - `ng-content` 表示组件内容占位符
  - `#tpl` 开头表示 ` `
- 对象相关，一般在类描述时
  - `[]` 表示属性
  - `()` 结尾表示方法
