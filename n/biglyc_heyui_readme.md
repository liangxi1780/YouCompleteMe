## Development

You need install [hey-cli](http://u.720life.cn/g/920c024f0b8c5aa5e32c4f88af4e6c9643bdb0c13d44d9e874e0ff88af164510c7d92fbf928a3a1f7e82e7ce290430dd 

```sh
#build
npm install
# build complete heyui
hey b
# build esm version
hey b -f hey.esm.js
# build lang js
hey b -f hey.lang.js

# develop
cd doc
npm install
hey dev

```
